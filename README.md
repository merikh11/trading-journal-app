# trading-journal-app
ژورنال ترید - اپلیکیشن ثبت معاملات
این اپلیکیشن برای ثبت و تحلیل معاملات طراحی شده است.

قابلیت‌ها:
✅ ثبت معاملات جدید
✅ محاسبه خودکار سود/زیان
✅ آمار کلی عملکرد
✅ تحلیل پیشرفته
✅ ذخیره محلی
نحوه استفاده:
1.
فایل index.html را در مرورگر باز کنید
2.
معاملات خود را اضافه کنید
3.
آمار و تحلیل‌ها را مشاهده کنید
نصب به عنوان اپلیکیشن:
در موبایل: منوی مرورگر > Add to Home Screen
برای APK: از PWA Builder استفاده کنید
فایل‌ها:
index.html: صفحه اصلی اپلیکیشن
manifest.json: تنظیمات PWA
README.md: این فایل راهنما
